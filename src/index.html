Theme

You can (and should) use themes to customize the look of the overlays generated by this tool. Themes should be placed in 'src/obs_src/themes' and require a short manifest file to be loaded by the interface. See the repository readme for more details.

You must click the "Set Theme" button in order to change the theme.
